# Business Hour Scheduling Logic (America/Sao_Paulo)
Implement a helper: computeNextCallbackDateTime(intakeTime)
Rules in America/Sao_Paulo:
- Working days: Monday to Friday
- Working hours: 09:00 to 18:00
- Default callback delay: 30 minutes (config DEFAULT_CALLBACK_DELAY_MINUTES=30)
- Duration not needed for Zoho task, but okay to include in description.

Algorithm:
1) Let now = intakeTime in America/Sao_Paulo
2) If weekend (Sat/Sun): set callback to next Monday 09:00
3) Else if time < 09:00: set callback to today 09:00
4) Else if 09:00 <= time <= 17:30: set callback = now + 30 minutes
5) Else (after 17:30): set callback to next business day 09:00
6) If callback time falls outside business hours, roll to next business day 09:00

Store resulting datetime as UTC in DB; send appropriate format to Zoho (respecting Zoho timezone expectations; if Zoho expects local date/time fields, convert accordingly).
