B. Benchmark Create / Edit Form (Exact Copy)
Section 1: Scope

Segment
Select the client segment this range applies to.

Region
State or macro-region where this benchmark is valid.

Contract Term
Length of the energy contract.

Helper text (below all three):
“These parameters define when ECOS will apply this reference range.”

Section 2: Reference Range

Lower Bound (R$/MWh)
Input the lower limit of the healthy reference range.

Upper Bound (R$/MWh)
Input the upper limit of the healthy reference range.

Helper text (important):
“A reference range is not a ‘best price’.
It defines where contracts are considered healthy, acceptable, or misaligned.”

Inline warning (if upper < lower):
“Upper bound must be greater than lower bound.”

Confidence Level
Low / Medium / High

Helper text under Confidence:
“Use High only when the data is recent and well-supported.
