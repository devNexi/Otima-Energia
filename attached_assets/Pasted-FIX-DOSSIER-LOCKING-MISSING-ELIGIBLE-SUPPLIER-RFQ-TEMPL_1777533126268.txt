Bug 2 — Genial Energia not showing as eligible supplier
What's wrong: The "Request Quotes" panel shows Prime Energy, ATMO Energia, CEMIG, Delantis Energias Renováveis — but NOT Genial Energia. The client is ASSOCIAÇÃO MAIS VIDA with distributor LIGHT RJ. According to the playbook data, Genial Energia covers Light + Enel RJ. So Genial should appear as eligible.
Root cause: The supplier eligibility query is not matching Genial's GD coverage to this client.
How to diagnose:
Step 1 — Check Genial's GD coverage in the database:
sqlSELECT s.name, gc.covered_states, gc.covered_distributors, gc.is_active
FROM supplier_gd_coverage gc
JOIN suppliers s ON s.id = gc.supplier_id
WHERE s.name LIKE '%Genial%';
Step 2 — Check the client's distributor value:
sqlSELECT distributor FROM client_dossiers WHERE client_id = <clientId>;
Step 3 — The problem is almost certainly a string mismatch. The dossier stores "LIGHT RJ" but the GD coverage stores "Light" or "Light RJ" (different case or format).
Fix — make distributor matching case-insensitive and partial:
Find the supplier eligibility query in server/routes.ts. Search for gd-eligible or gd_coverage or wherever eligible suppliers are filtered.
The matching logic should be:
typescript// WRONG — exact match:
coveredDistributors.includes(clientDistributor)

// RIGHT — case-insensitive partial match:
function distributorMatches(covered: string[], clientDist: string): boolean {
  if (!clientDist) return false;
  const normalized = clientDist.toLowerCase().trim();
  return covered.some(d => {
    const norm = d.toLowerCase().trim();
    // Check both directions — "Light" matches "LIGHT RJ" and vice versa
    return normalized.includes(norm) || norm.includes(normalized);
  });
}
Apply this in the eligibility filter:
typescript// When filtering suppliers for RFQ:

  // Check if supplier has a playbook matching the deal's product line

    pb.status === 'ACTIVE' &&
    pb.productLines.some(pl => {
      if (dealTrackType === 'GDL') return pl === 'GD';
      if (dealTrackType === 'ACL') return pl === 'ACL';
      return false;
    })
  );
  if (!matchingPlaybook) return false;

  // For GD deals, also check distributor coverage

    const coverage = gdCoverages.find(gc =>
      gc.supplierId === supplier.id && gc.isActive

    if (!coverage) return false;

    // Delantis has national coverage — always eligible
    if (coverage.coveredStates?.includes('ALL')) return true;

    // Check distributor match (case-insensitive, partial)
    return distributorMatches(coverage.coveredDistributors || [], clientDistributor);
  }

  return true; // ACL — if playbook matches, supplier is eligible
});
